SystemInformation

data class SystemInformation(var versionInfo: VersionInfo? = null, var platform: String? = null) : OverlayNode

Represents different system information that are used in the PythonValueEvaluator to evaluate expressions, such as sys.platform and sys.version_info.

We model this as an overlay node so we can access this information later in the graph.
