extractScope

fun extractScope(node: Node, scope: Scope? = currentScope): Pair<Scope?, Name>

This function extracts a possible scope out of a Name, e.g. if the name is fully qualified. This also resolves possible name aliases (e.g. because of imports). It returns a pair of a scope (if found) as well as the name, which is possibly adjusted for the aliases.

Note: Currently only fully qualified names are properly resolved. This function will probably return imprecise results for partially qualified names, e.g. if a name A inside B points to A::B, rather than to A.
