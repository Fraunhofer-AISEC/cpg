constInitializers

var constInitializers: MutableMap<Int, GoStandardLibrary.Ast.Expr>

The current initializers in a list representing the different "columns". For example in the following code:

const (
  a, b = 1, 2
  c, d
  e, f = 4, 5
)
Content copied to clipboard

The current list of initializers would first be (1,2) until a new set of initializers is declared in the last spec. The key corresponds to the "column" of the variable (a=0,b=1).
