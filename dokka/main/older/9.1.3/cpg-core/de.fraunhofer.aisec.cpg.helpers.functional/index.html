Lattice

interface Lattice<T : Lattice.Element>

A lattice is a partially ordered structure of values of type T. T could be anything, where common examples are sets, ranges, maps, tuples, but it can also have random names and a new data structure which only make sense in a certain context. T depends on the analysis and typically has to abstract the value for the specific purpose.
