TypeObserver

interface TypeObserver

A TypeObserver can be used by its implementing class to observe changes to the HasType.type and/or HasType.assignedTypes of a Node (that implements HasType). The implementing node can then decide if and how to propagate this type information to itself (and possibly to others). Examples include modifying the incoming type depending on an operator, e.g., in a UnaryOperator expression. Changes to HasType.type will invoke typeChanged, changes to HasType.assignedTypes will invoke assignedTypes.
