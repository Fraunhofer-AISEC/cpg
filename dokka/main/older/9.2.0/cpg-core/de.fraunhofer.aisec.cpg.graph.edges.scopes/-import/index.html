Import

class Import(start: Scope, end: NamespaceScope, var declaration: ImportDeclaration? = null) : Edge<NamespaceScope>

This edge represents the import of a NamespaceScope into another Scope. The style of import (e.g., whether only a certain symbol or the whole namespace is imported) is determined by the declaration.
