FileOperation

abstract class FileOperation(underlyingNode: Node, file: File) : Operation

All Files inherit from this class. This makes the file field available for FileOperation, resulting in easier to read queries (one can use FileOperation.file instead of Operation.concept). There is no logic involved - just a simple forwarding of the field.
